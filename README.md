# Limpieza de Datos de Usuarios

## Descripción
Proyecto de limpieza y validación de datos de usuarios utilizando Python. Implementa procesos de estandarización de nombres, conversión de tipos y preparación de datasets para análisis de comportamiento de compra.

## Características
- **Normalización de Texto**: Limpieza de espacios y estandarización de formatos en nombres.
- **Validación de Tipos**: Conversión y verificación de datos numéricos (edades).
- **Manejo de Errores**: Implementación de bloques try-except para datos corruptos.
- **Estructuración de Datos**: Organización de listas y preparación para análisis.

## Tecnologías
- Python 3
- Jupyter Notebook
